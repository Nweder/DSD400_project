STARTUP AV SERVER:
Initierar server med: python manage.py runserver {port=8000}
Synkronisera servern med databas: python manage.py migrate

urls.py finns listan urlspattern som innehåller link till våra sidor.
Dessa hämtas från funktioner I views.py

I mappen 'templates' hittar vi våra .html
i mappen 'static' hiyttar vi .css .js
